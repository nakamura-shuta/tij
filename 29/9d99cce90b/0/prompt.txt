# Phase 29: Rebase revset 入力の実装

## Context

現在の tij rebase は単一 change_id のみ source に指定可能。
jj の `-r`, `-s`, `-b` フラグは revset を受け付けるため、`jj rebase -r 'A|B' -d C` のような複数リビジョン一括 rebase が可能。

本実装では RebaseSelect モード中に `:` キーで revset テキスト入力を追加し、
Revision/Source/Branch の 3 モードで revset 文字列を source に指定可能にする。
InsertAfter/InsertBefore は対象外。

**2c. `handle_rebase_revset_input_key()` 新規メソッド**

`handle_text_input()` は使わない（Esc の遷移先が `Normal` ではなく `RebaseSelect`）。
独立ハンドラ:

**セマンティクス定義:**
- **Enter（確定）**: 入力内容で revset を設定。空入力は revset モード解除 + source を現在選択中の change_id に復元。
- **Esc（キャンセル）**: 今回の入力を破棄し、**revset 設定ごと全解除**。source を現在選択中の change_id に復元し、`use_revset = false` にする。
  理由: Esc は「やめる」操作であり、中途半端な状態を残すべきでない。revset を再編集したい場合はもう一度 `:` を押せばよい。

```rust
fn handle_rebase_revset_input_key(&mut self, key: KeyEvent) -> LogAction {
    match key.code {
        k if k == keys::ESC => {
            // Esc: 入力破棄 + revset モード全解除
            self.input_buffer.clear();
            self.rebase_use_revset = false;
            // source を現在選択中 change_id に復元
            if let Some(change) = self.selected_change() {
                self.rebase_source = Some(change.change_id.clone());
            }
            self.input_mode = InputMode::RebaseSelect;
            LogAction::None
        }
        k if k == keys::SUBMIT => {
            let revset = std::mem::take(&mut self.input_buffer);
            if revset.is_empty() {
                // 空 Enter: revset モード解除 + source を現在選択中 change_id に復元
                self.rebase_use_revset = false;
                if let Some(change) = self.selected_change() {
                    self.rebase_source = Some(change.change_id.clone());
                }
            } else {
                self.rebase_source = Some(revset);
                self.rebase_use_revset = true;
            }
            self.input_mode = InputMode::RebaseSelect;
            LogAction::None
        }
        KeyCode::Char(c) if !key.modifiers.intersects(KeyModifiers::CONTROL | KeyModifiers::ALT) => {
            self.input_buffer.push(c);
            LogAction::None
        }
        KeyCode::Backspace => {
            self.input_buffer.pop();
            LogAction::None
        }
        _ => LogAction::None,
    }
}
```

> **Note**: `self.selected_change()` は immutable borrow なので `.clone()` が必要。
> `rebase_source` は元々 `start_rebase_mode_select()` で設定された change_id なので、
> 復元先は「現在カーソル位置の change」で正しい（ユーザーが destination を選んでいる最中なので）。
